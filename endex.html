O que é a World Wide Web?

A World Wide Web (WWW), ou simplesmente Web, é um sistema de informações interligadas que são acessadas através da internet. Foi inventada em 1989 por Tim Berners-Lee enquanto trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear).

Como funciona a Web?

A Web funciona através de uma arquitetura cliente-servidor. Um usuário utiliza um software chamado navegador (como o Google Chrome ou Firefox) para acessar páginas da Web armazenadas em servidores. A comunicação entre o navegador e o servidor ocorre por meio de um protocolo chamado HTTP (HyperText Transfer Protocol).

Principais Tecnologias da Web

As páginas da Web são construídas utilizando três tecnologias principais:

  • HTML (HyperText Markup Language): A linguagem usada para estruturar o conteúdo da Web.
  • CSS (Cascading Style Sheets): Usado para estilizar e definir o layout das páginas da Web.
  • JavaScript: Uma linguagem de programação que adiciona interatividade e dinamismo às páginas.

Exemplo de código HTML

Abaixo está um exemplo de código HTML básico:

<!DOCTYPE html>
<html>
<head>
    <title>Minha Primeira Página Web</title>
</head>
<body>
    <h1>Olá, Mundo!</h1>
    <p>Este é um exemplo de uma página HTML simples.</p>
</body>
</html>
